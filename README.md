# Excelsior

Includes patched DSDT and SSDT extras

+ 1.0


Hardware

+ EVGA X299 Micro 2
+ 16 GB (2x8) Corsair vengance 3200mhz
+ Intel i9 7900x @ 4.9GHZ
+ Samsung 970 EVO Plus 500GB NVME
+ ASRock AMD 5700XT 8GB
